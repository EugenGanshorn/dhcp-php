addOption()

addOption(integer  $option, integer  $length, mixed  $details = null)

Adds an option with given data to the list.

This should be used to add an option from the socket data; not manually creating an option. To manually add a new/replace existing option, use DHCPPacket\

Option number. If it's not defined in \DHCP\Options\DHCPOptionX, it will be skipped.
