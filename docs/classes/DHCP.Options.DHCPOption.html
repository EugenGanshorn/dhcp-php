__construct()

__construct(mixed  $length = null, mixed  $details = false, \Psr\Log\LoggerInterface  $logger = null)

If $length is provided, the basic details about the data from the packet will be checked.

Parameters to the constructor should only be passed when creating the option from socket data. If you want to create an option manually, don't pass anything here, use specific set() methods to modify the option's values.
